#### Notes：
* 图片请放在/images/文件夹，最好各自建立对应的目录分开管理
* 引入外部css或者js，请放入/assets/目录下
* /_steps/下文件用英文名，不要有空格，单词之间连接用'-'（比如“config-of-jekyll”），这样生成的url中就不会有中文字符了
